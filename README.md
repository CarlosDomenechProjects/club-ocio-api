# Club Ocio API

Backend del SaaS para la gestión de un club de ocio con actividades y membresías.

## Funcionalidad (visión general)

- Registro e inicio de sesión de usuarios.
- Gestión de roles:
  - SUPERADMIN
  - ADMIN
  - USER
- Gestión de usuarios:
  - Alta
  - Inactivación
  - Borrado definitivo
- Gestión de membresías y actividades.
- Exposición de una API REST para el frontend (Next.js).

## Stack previsto

- Node.js
- TypeScript
- NestJS
- Base de datos SQL (por ejemplo PostgreSQL)
- ORM (por decidir: TypeORM / Prisma)
